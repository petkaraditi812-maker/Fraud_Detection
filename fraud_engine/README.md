# Fraud Detection System

## Overview
End-to-end fraud detection pipeline integrating MongoDB, feature engineering, rule-based scoring, machine learning, PostgreSQL storage, and real-time API.

## Tech Stack
- Python
- MongoDB
- PostgreSQL
- scikit-learn
- Flask
- Power BI (optional)

## Features
- Risk-based scoring
- Logistic Regression model
- Real-time fraud API
- SQL fraud storage
